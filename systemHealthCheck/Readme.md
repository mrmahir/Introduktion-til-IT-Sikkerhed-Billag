Et simpelt check på nogle få systemadministrative informationer som FireWall or hvilke porte man har åbent.
Kunne være ret cool at bruge efter man måske har leget med portforwarding og man lige gerne vil være sikker på nogle vigtige porte er lukkede.
Det kan selvfølgelig vidreudvikles til et mere omfattende program, men jeg forstod formålet med denne at;
Vise at jeg kan lave Bash-, PowerShell og Python scripts, så jeg lavede alle i en som både virker på Linux og Windows, hvor et Python Script checker, hvilken OS man køre på og derefter køre enten Linux audit eller Windows Audit.
